# 🚗 Car Features and Pricing Pattern Analysis

## 📘 Project Overview
This project performs an in-depth data analysis of global car models using Python.
It explores how engine capacity, performance, and fuel type influence car pricing and market trends.

## 🎯 Objectives
- Identify key factors affecting car prices  
- Compare brand-wise and fuel-type trends  
- Analyze adoption of electric vehicles (EVs)  
- Generate actionable insights for the automobile industry  

## 📊 Dataset Information
The dataset includes car specifications such as company, model, engine/battery capacity, horsepower, price, fuel type, and performance metrics.

## 🧠 Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🧩 Steps
1. Data Cleaning and Preprocessing  
2. Exploratory Data Analysis (Univariate, Bivariate, Multivariate)  
3. Visualization and Insights  

## 📈 Outcome
Data-driven insights on pricing trends, performance analysis, and fuel technology adoption across global car brands.
